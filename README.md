# Galliate project

Copy of [Nubank app](https://nubank.com.br)


## How to test

1. ```$ git clone```
1. ```cd galliate```
1. ```$ yarn```
1. start android or ios emulator
1. ```$ npx react-native run-android``` or ```$ npx react-native run-ios```
